To display the message (we can use label) <label>Welcome to HTML Controls</label>
TextBox (or) Input TextBox <input type=text name=myname id=myname required placeholder="Enter Your Name">
MobileNumber (or) Number TextBox <input type=number name=mymno id=mymno required placeholder="Enter Your Mobile No">
Email TextBox <input type=email name=myemail id=myemail required placeholder="Enter Your Email id">
To create the Calendar control <input type=date name=bdate id=bdate>
To create the Timer control <input type=time name=btime id=btime>
To create the drop down list <select name=mycolor id=mycolor> <option>Select</option> <option>Red</option> <option>Green</option> <option>Orange</option> </select>
RadioButton (Male and Female button for select any one option) Male Female <input type=radio name=rb id=rb value="Male">Male <input type=radio name=rb id=rb value="Female">Female
CheckBox (To select multiple option) HTML CSS Javascript PHP SQL Python DJango <input type=checkbox name=cb id=cb value="HTML">HTML <input type=checkbox name=cb id=cb value="CSS">CSS <input type=checkbox name=cb id=cb value="Javascript">Javascript <input type=checkbox name=cb id=cb value="PHP">PHP <input type=checkbox name=cb id=cb value="SQL">SQL <input type=checkbox name=cb id=cb value="Python">Python <input type=checkbox name=cb id=cb value="DJango">DJango
To create the button Or <button>Submit</button> <button>Login</button>
